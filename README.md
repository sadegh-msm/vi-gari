# vichare
my neovim config from scratch

## binding
`<leader>` is space

### basic vim
`<leader>pv` for go to netrw
`<C-q>` for selecting rectangular

### harpoon
`<leader>a` add file to harpoon
`<C-e>` harpoon quick menu 
`<C-h>` file 1
`<C-t>` file 2 
`<C-n>` file 3
`<C-s>` file 4

### telescope
`<leader>ff` find files
`<leader>ps` grep inside file
`<C-p>` find inside git
